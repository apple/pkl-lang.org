  • expand_morelink
    structTags: StructTagsSource

    Additional struct tags for the annotated Pkl property.

    All properties also receive an implicit pkl struct tag.

    The mapping key becomes the struct tag's key. The mapping value receives the Pkl property name as an argument, and the return value becomes the struct tag's value.
