  • expand_morelink
    multilineThreshold: IntSource

    The number of elements in a Mapping or Listing to render them multiline.

    The default value of 2 means a collection with a single element will be rendered in the inline style, although any nested collections may be rendered in the multiline style.

    Note that Map and List are always rendered in the inline style.
