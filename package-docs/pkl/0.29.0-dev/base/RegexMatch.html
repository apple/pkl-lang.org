  • expand_morelink
    groups: List<RegexMatch?>

    The capturing group matches within this match.

    If this is already a capturing group match, returns the empty list. Otherwise, returns a list of length regex.groupCount + 1, where groups[0] contains the entire match (for consistency with other regex APIs), and groups[1] to groups[regex.groupCount] contain capturing group matches. If a capturing group did not produce a match, the corresponding RegexMatch element is null.
