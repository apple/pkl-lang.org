external class Float extends Number

A 64-bit floating-point number conforming to the IEEE 754 binary64 format.

Known subtypes:
Known usages:
All versions:

The following binary operators are supported for Numbers:

-a     // negative
a + b  // addition
a - b  // subtraction
a * b  // multiplication
a / b  // division
a ~/ b // integer division
a % b  // remainder
a ** b // exponentiation

Tip: To allow transparent use of Float and Int, use Number instead of Float in type annotations.
